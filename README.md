اطلاعات تکنیکی

این بخش نحوه عملکرد عملکردهای مختلف برنامه را توضیح می دهد و تجزیه و تحلیلی از عملیات آنها ارائه می دهد.

گزارش دسته جمعی

برای دستیابی به عملکرد گزارش انبوه، برنامه از فرهنگ لغت report_headers استفاده می کند. این هدرها برای ایجاد درخواست های HTTP و ارائه اطلاعات لازم به سرور ضروری هستند. در اینجا هدرهای مورد استفاده آمده است:

"Accept": "*/*",
"Accept-Encoding": "gzip, deflate",
"Accept-Language": "tr-TR,tr;q=0.8,en-US;q=0.5,en;q=0.3",
"Cache-Control": "no-cache",
"Connection": "keep-alive",
"Content-Type": "application/x-www-form-urlencoded",
"DNT": "1",
"Host": "help.instagram.com",
"Origin": "help.instagram.com",
"Pragma": "no-cache",
"Referer": "https://help.instagram.com/contact/497253480400030",
"TE": "Trailers",
این هدرها در درخواست HTTP برای اطمینان از ارتباط مناسب و ارائه جزئیات لازم به API مرکز راهنمای اینستاگرام گنجانده شده‌اند .

کد برنامه با ارسال یک درخواست POST به URL مشخص شده با API فرم تماس مرکز راهنمایی اینستاگرام تعامل دارد.

نیروی بی رحم

تابع insta_bruteforce با استفاده از یک نام کاربری ارائه شده و یک لیست کلمه از رمزهای عبور، یک حمله brute force به حساب اینستاگرام انجام می دهد. در اینجا یک نمای کلی از نحوه عملکرد آن است:

این تابع سه پارامتر دارد: نام کاربری (اکانت هدف اینستاگرام)، فهرست کلمات (نام فایل فهرست کلمات رمز عبور) و vpn (پرچم استفاده از VPN).

فایل wordlist را می خواند و رمزهای عبور را در حافظه ذخیره می کند.

برای هر رمز عبور در لیست کلمات، تابع مراحل زیر را انجام می دهد:

یک درخواست POST را با نام کاربری و رمز عبور ارائه شده به API ورود به اینستاگرام ارسال می کند.
اگر پاسخ نشان دهنده ورود موفقیت آمیز باشد، رمز عبور را نمایش می دهد و برنامه را خاتمه می دهد.
اگر تلاش برای ورود ناموفق بود، به رمز عبور بعدی منتقل می شود.
اگر یک استثنا در طول فرآیند رخ دهد (به عنوان مثال، خطای شبکه)، به رمز عبور بعدی منتقل می شود.

اگر پرچم VPN روی True تنظیم شده باشد، سعی می کند آدرس IP را تغییر دهد.

روش دوم، در حال حاضر در حال استفاده است

  try:
    L.login(USER, PASSWORD)
    return 1
  except Exception as e:
    if "Checkpoint" in str(e):
      return 1
    elif "incorrect" in str(e):
      return 0
    elif "blocked" in str(e):
      return 0
    else:
      return 0
این تابع بررسی می‌کند که آیا رمز عبور دقیق است یا نه، اگر تأیید درخواست شود، رمز عبور دقیق است اما کاربر از تأیید دو مرحله‌ای استفاده می‌کند، از سوی دیگر اگر تأیید درخواست نشود رمز عبور دقیق است.

خطاها

این بخش خطاهای رایجی را که ممکن است در برنامه رخ دهد مورد بحث قرار می دهد و علل اساسی آنها را بررسی می کند.

خطای 0x00: حرف به جای عدد عددی

این خطا زمانی رخ می دهد که به اشتباه یک حرف به جای مقدار عددی مورد نیاز درج شود. معمولاً به دلیل مشکلات مربوط به ورودی کاربر یا پردازش داده که در آن یک کاراکتر غیر عددی به جای یک مقدار عددی معتبر ارائه می‌شود، اتفاق می‌افتد.

خطای 0x01: شماره خارج از محدوده

خطای 0x01 زمانی ایجاد می شود که عددی خارج از محدوده مورد انتظار در یک تعیین کننده یا محدوده خاص وارد شود. این خطا معمولاً به دلیل ورودی کاربر یا یک محاسبه اشتباه که منجر به مقداری فراتر از محدوده مجاز می شود ایجاد می شود.

خطای 0x02: عدد نادرست در تعیین کننده یا محدوده

وقوع خطای 0x02 نشان می دهد که عددی خارج از تعیین کننده یا محدوده تعیین شده وارد شده است. این خطا معمولاً زمانی رخ می دهد که عددی ارائه می شود که خارج از مرزهای مورد انتظار قرار می گیرد و می تواند به دلیل ورودی کاربر یا خطا در محاسبات ایجاد شود.

خطای 0x03: استفاده از VPN در ویندوز

هنگام تلاش برای استفاده از VPN در سیستم ویندوز با خطای 0x03 مواجه می شود. این خطا هنگام تلاش برای ایجاد اتصال VPN در سیستم عامل ویندوز، مشکلات سازگاری یا پیکربندی نادرست را نشان می دهد.

قابل اعتماد بودن

می توانید با تست کردن آن بر روی پروفایل خود بررسی کنید که برنامه کار می کند ، لیست کلمات را باز کرده و رمز پروفایل خود را در وهله اول قرار دهید، سپس برنامه را راه اندازی کنید و نام nckname پروفایل اینستاگرام خود را وارد کنید و خواهید دید که پسورد صحیح ظاهر می شود.

خطای 0x01: شماره خارج از محدوده

خطای 0x01 زمانی ایجاد می شود که عددی خارج از محدوده مورد انتظار در یک تعیین کننده یا محدوده خاص وارد شود. این خطا معمولاً به دلیل ورودی کاربر یا یک محاسبه اشتباه که منجر به مقداری فراتر از محدوده مجاز می شود ایجاد می شود.

خطای 0x02: عدد نادرست در تعیین کننده یا محدوده

وقوع خطای 0x02 نشان می دهد که عددی خارج از تعیین کننده یا محدوده تعیین شده وارد شده است. این خطا معمولاً زمانی رخ می دهد که عددی ارائه می شود که خارج از مرزهای مورد انتظار قرار می گیرد و می تواند به دلیل ورودی کاربر یا خطا در محاسبات ایجاد شود.

خطای 0x03: استفاده از VPN در ویندوز

هنگام تلاش برای استفاده از VPN در سیستم ویندوز با خطای 0x03 مواجه می شود. این خطا هنگام تلاش برای ایجاد اتصال VPN در سیستم عامل ویندوز، مشکلات سازگاری یا پیکربندی نادرست را نشان می دهد.
